# 18반 CS 스터디

# 규칙
1. 각자 담당한 주제에 대한 발표 자료 준비
2. 준비한 발표 자료는 해당 주제 폴더 내부에 "n주차_발표 범위 제목_이름" 형태로 올리기
   - 예 : 1주차_운영체제의역할과구조_김준혁
3. 커밋 메세지는 자유
