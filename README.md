## 📌 규칙
1. **발표 자료 준비:** 각자 담당한 주제에 대한 발표 자료를 준비합니다.  
2. **파일 업로드:**  
   - 해당 주제 폴더에 **"n주차_발표 범위 제목_이름"** 형태로 올려주세요.  
   - 예시: `1주차_운영체제의역할과구조_김준혁`  
   - 파일 형식은 **.md(권장)** 또는 **.pdf** 중 선택 가능합니다.  
   - 이미지가 필요하다면 **img 폴더**를 만들어 사용하세요! 📂  
3. **질문 준비:** 각자 맡은 범위의 질문 **5개**씩 준비해주세요.  
4. **커밋 메시지:** 자유롭게 작성합니다! 😄
